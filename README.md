# 🎓 Student Management System in C

A menu-driven Student Management System developed in C that allows users to manage student records efficiently using structures and file handling. The program stores data permanently in files, ensuring records remain available even after the program is closed.

## ✨ Features

* ➕ Add New Student Record
* 🔍 Search Student by ID
* ✏️ Update Existing Record
* ❌ Delete Student Record
* 📋 Display All Student Records
* 💾 Permanent Data Storage using File Handling
* 🎯 Menu-Driven User Interface

## 🛠 Technologies Used

* C Programming Language
* Structures (`struct`)
* File Handling (`fopen`, `fclose`, `fread`, `fwrite`)
* Functions
* Conditional Statements
* Loops

## 📚 Concepts Covered

* Data Structures using Structures
* File Management
* CRUD Operations
* Searching Records
* Modular Programming
* Data Persistence

## 📂 Student Record Fields

Each student record contains:

* Student ID
* Student Name
* Age
* Course
* Marks
